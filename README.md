# react-hexagonal-sample
- Sample project to describe React + Hexagonal architecture

# Models

```
- Todo
  - properties
    - id: string
    - title: string
    - done: boolean
    - tagIds: string[]
    - createdAt: Datetime
    - updatedAt: Datetime

- Notification
  - properties
    - id: string
    - message: string
    - createdAt: Datetime
    - updatedAt: Datetime

- Tag
  - properties
    - id: string
    - name: string
    - color: string
    - createdAt: Datetime
    - updatedAt: Datetime
```
